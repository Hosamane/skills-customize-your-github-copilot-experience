# 📘 Assignment: Interactive Data Storytelling with Plotly

## 🎯 Objective

Create an interactive, web-ready data story using Plotly. Students will load a real dataset, build interactive visualizations, and assemble a short HTML report that combines figures and explanatory captions.

## 📝 Tasks

### 🛠️ Data Selection & Preparation

#### Description
Choose a dataset (provided `data.csv` or another CSV), clean and prepare it for visual exploration.

#### Requirements
Completed work should:

- Load the provided CSV and display the first 5 rows
- Handle missing values and parse dates where relevant
- Provide a short dataset summary (columns and basic statistics)


### 🛠️ Interactive Visualizations

#### Description
Create multiple interactive Plotly charts that allow users to explore the data (hover, zoom, filter).

#### Requirements
Completed work should:

- Produce at least three distinct interactive charts (e.g., time series, scatter with hover, bar with category breakdown)
- Add informative hover text and axis labels
- Save each figure to an HTML fragment or file


### 🛠️ Data Story & HTML Report

#### Description
Combine charts and concise captions into a single `report.html` that presents a short, data-driven narrative.

#### Requirements
Completed work should:

- Assemble an HTML report embedding the interactive figures
- Include 2–4 short captions explaining the main insights
- Provide instructions for opening and sharing the report


### 🛠️ Reproducibility

#### Description
Make it easy for others to recreate your results.

#### Requirements
Completed work should:

- Include `requirements.txt` with key dependencies
- Include run instructions (one or two commands) in the README

---

Optional extensions:

- Add filtering controls (Dash or Streamlit)
- Use map visualizations for geospatial data
- Add small animations or transitions
