
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game to practice string manipulation, control flow, and user input handling in Python.

## 📝 Tasks

### 🛠️ Game Mechanics

#### Description
Implement the core Hangman gameplay: selecting a secret word, accepting letter guesses, tracking attempts, and revealing progress.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Prompt the player to guess single letters and update the displayed word progress (e.g., _ a _ _ _)
- Track and display the number of incorrect guesses remaining
- End the game when the word is fully guessed or attempts are exhausted
- Show clear win/lose messages


### 🛠️ Gameplay and UX

#### Description
Improve the player experience with input validation and helpful messages.

#### Requirements
Completed program should:

- Ignore repeated guesses and inform the player
- Validate input so only single alphabetic characters are accepted
- Optionally display the list of guessed letters
- Include at least one example of running the game in the README (sample interaction)
